Welcome to the After Effects basics website.
Here, you will learn everything you need to navigate the programs UI and how to compose simple videos.

Description

This website aims to be a comprehensive learning hub tailored for new users aspiring to master Adobe After Effects.
It will feature a user-friendly interface and a curated curriculum, starting with the fundamentals and user interface.
The content will include step-by-step tutorials ranging from simple concepts to practical exercises that build essential skills,
illustrative videos, and real-world project examples to reinforce learning.
A page will be dedicated to learning about me and my background in the use of this program,
and a curated list of trusted resources to help with learning the software.
By simplifying the learning process and focusing on fundamental principles, the website will strive to break down the complexities of After Effects,
making it an accessible and enjoyable learning experience,
ultimately equipping new users with the skills and confidence to create stunning motion graphics and visual effects.
